# **Event Registration**
## Members: Batac, Bunyad, Dela Torre, Fajardo, Francisco

An Event Registration System is a web-based application designed to simplify the process of managing event participants and registrations. This system allows users to register for an event by entering their personal information through an online form, while administrators can manage participant records efficiently. The project uses PHP and MySQL to perform essential CRUD operations such as adding, viewing, updating, and deleting registration data.

The main purpose of this project is to replace manual registration methods with a faster, more organized, and accurate digital system. By automating the registration process, the system reduces paperwork, minimizes errors, and improves data management. This project also demonstrates the practical application of web development concepts, database management, and server-side programming using PHP.
